# Ahmed Mahmoud Abbas — Portfolio

Personal portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

**Live:** [portfolio-azure-beta-0b60zh90oz.vercel.app](https://portfolio-azure-beta-0b60zh90oz.vercel.app)

---

## Stack

- **Framework:** Next.js 14 (App Router, static export)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + custom CSS animations
- **Icons:** lucide-react
- **Fonts:** Geist Mono, Inter
- **Deployment:** Vercel (auto-deploy from `main`)

## Sections

- Hero — tagline, CV download, GitHub link
- About — prose + terminal block
- Featured Project — ACR-QA (graduation project)
- Projects — 5 featured cards + 7 compact entries
- Experience — DEPI, NBK Egypt, Emkay Foods
- Skills — Languages, Backend, DevOps, Cloud & Infra, ML & AI, Mobile & Web
- Recognition — ICPC ECPC 2024 Honorable Mention
- Currently — what I'm working on outside of code
- Contact — email, GitHub, LinkedIn, LeetCode

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Pushes to `main` trigger automatic Vercel deployments.

```bash
git push origin main
```
